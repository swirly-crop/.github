# 🐉 Swirly Crop

> Low-level systems engineering --from bare metal to the browser.

## About

Swirly Crop is a systems engineering company that works at the intersection
of hardware and software. We write the code that makes everything else run.

## What We Do

- 🔧 **CPU Drivers** - Low-level driver development across x86, ARM, and 
  embedded architectures. We write close to the metal so your stack doesn't 
  have to worry about what's beneath it.

- ⚡ **GPU Drivers** - From kernel-mode drivers to user-space libraries, we 
  handle the full pipeline. Parallel execution, memory management, shader 
  compilation - we've been in the weeds so you don't have to be.

- 🌐 **WebGL** - Bridging GPU hardware and the browser. Real-time graphics, 
  optimized draw calls, and cross-device consistency for web-based rendering 
  pipelines.

- 🖥️ **Server Maintenance** - Infrastructure that stays up. We treat uptime 
  and reliability as engineering problems - monitoring, patching, tuning, 
  and keeping your systems predictable under load.
